## 🚀 خطوات التشغيل

### التثبيت والتشغيل
```bash
# 1. تثبيت التبعيات
npm install

# 2. إعداد المتغيرات البيئية
cp .env.example .env.local

# 3. إعداد قاعدة البيانات
npx prisma generate
npx prisma db push

# 4. تشغيل المشروع
npm run dev
```

### النشر مع Docker
```bash
# بناء وتشغيل مع Docker Compose
docker-compose up --build

# أو بناء صورة Docker منفصلة
docker build -t maraseem-home-decor .
docker run -p 3000:3000 maraseem-home-decor
```
